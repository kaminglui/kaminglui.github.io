Labs

How the labs fit together

Seven interactive labs covering the math and methods behind modern machine learning. They build on each other — skim the tier you already know, spend time on the one you don't. Hover any chip to see what a term means; click to jump straight to that section.

Tier 1 Foundations

Math Foundations

Optimisation · Linear algebra · PCA

Calculus, linear algebra, and PCA — what every ML method reuses. Interactive GD/SGD/SA on a 2D landscape, and a live PCA of a 2D scatter.

  • argmin · Lagrangian
  • GD / SGD / momentum
  • eigenvalues · SVD
  • universal approx

Fourier Epicycles

Signals, DFT, FFT

Decompose any signal into rotating circles. DFT, FFT (Cooley-Tukey), 2D Fourier, and the SDE bridge that shows up again in Diffusion.

  • DFT · FFT
  • Cooley-Tukey matrix proof
  • 2D Fourier
  • Convolution theorem
builds on
Tier 2 Core ML

ML Lab

Supervised + unsupervised, from first principles

Linear regression via closed-form OLS, logistic regression via GD on cross-entropy, SVM with margin lines + support vectors, K-means and Gaussian-mixture clustering via EM. The full math — MLE, MAP, Lagrangian duals, kernels, bias–variance, regularisation.

  • squared / hinge / cross-entropy loss
  • MLE · MAP · EM
  • K-means · GMM · responsibilities
  • kernels · ridge / lasso
builds on
Tier 3 Modern deep learning

RL Lab

Bandits → MDP → actor-critic

Agents that learn from reward. Bandits, value iteration on a 1D chain, Monte Carlo darts, Bayesian coin update, and nine KaTeX theory panels ending in a method clustering map.

  • ε-greedy · UCB · Thompson
  • value / policy iteration
  • TD · SARSA · Q-learning
  • actor-critic (PPO / SAC)

Transformer Lab

Live attention pipeline

Type a sentence and watch it turn into tokens, Q/K/V projections, an attention heatmap, and a blended context vector — every number computed live.

  • tokens · embeddings
  • Q · K · V
  • softmax attention
  • context vectors

Diffusion Lab

Noise → data

Watch a 2D particle cloud dissolve into Gaussian static and re-assemble using the analytic score of the target mixture. VAEs, DDPM, score matching, SDEs, flow matching.

  • KL · ELBO · VAE
  • DDPM · score matching
  • VP / VE SDE · PF-ODE
  • flow matching · consistency

Analog circuit simulator

Hardware / EE side of the site — a full in-browser SPICE-style simulator for resistors, caps, inductors, op-amps, diodes, BJTs, MOSFETs. Independent of the ML track above.
